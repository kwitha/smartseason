 # SmartSeason
SmartSeason is a full stack (Laravel(backend)+ Vue.js(frontend) ) application for managing fields,assigning agents,and tracking field progress over time.

It is designed for admins to monitor farms and for agents to update field conditions.

## Tech Stack
- Backend:Laravel
- Fronted:Vue.js
- API:RESTful laravel Api
- Auth:Token-based authentication(sanctum)
- Database:MySql

## Features
### Admin
- Creates and manage fields
- Assign agents to fields
- Monitor field progress and updates
- view all system data

### Agent
- View assigned fields
- Update field stages
- Add field notes
